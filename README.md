# 서울 AI 허브 미팅룸 예약 자동화

서울 AI 허브(seoulaihub.kr) 시설 예약 페이지를 자동화한 대화형 CLI 스크립트입니다.

---

## 개요

- Playwright 기반 브라우저 자동화로 로그인부터 예약 신청까지 처리
- 대화형 인터페이스 — 번호 선택으로 건물·시설 선택, 입력 오류 시 재입력 안내
- 예약 현황 조회 (특정 날짜 **전체 건물** 시설 / 이번 달 우리 예약)
- 예약 취소 — **전체 건물** 대상, 현재 이후 예약만 시간순으로 표시
- `.env` 파일로 로그인 정보 관리

---

## 구조

```
meetingroom_reservaitior/
├── reserve.py          # 진입점 (대화형 메뉴)
├── modules/
│   ├── config.py       # 상수 (URL, 사이트 ID 등)
│   ├── auth.py         # 로그인
│   ├── book.py         # 예약
│   ├── query.py        # 조회
│   ├── cancel.py       # 취소
│   └── outlook.py      # 아웃룩 캘린더 연동 (Microsoft Graph API)
└── .env                # 로그인 정보 (SEOULAIHUB_ID, SEOULAIHUB_PW, OUTLOOK_CLIENT_ID)
```

**주요 함수**

| 모듈 | 함수 | 설명 |
|---|---|---|
| `modules/book.py` | `reserve()` | 예약 실행 (STEP1 → STEP2 → 신청) |
| `modules/book.py` | `get_facilities()` | 사이트별 시설 목록 동적 조회 |
| `modules/query.py` | `list_reservations()` | 예약 목록 조회 (전체 건물, 월별 / 날짜 지정) |
| `modules/cancel.py` | `cancel_reservation()` | 예약번호로 취소 |
| `modules/outlook.py` | `create_event()` | 아웃룩 캘린더 일정 등록 (참가자 초대 포함) |
| `modules/outlook.py` | `cancel_event()` | 아웃룩 캘린더 일정 취소 |

## 환경 설정

```bash
bash setup.sh
```

또는 직접 설치:

```bash
pip install -r requirements.txt
playwright install chromium
```

`.env` 파일 생성:
```
SEOULAIHUB_ID=아이디
SEOULAIHUB_PW=비밀번호
OUTLOOK_CLIENT_ID=애저_클라이언트_ID   # 아웃룩 연동 시 필요
```

## 아웃룩 캘린더 연동 설정 (선택)

예약/취소 시 아웃룩 캘린더에 자동으로 일정을 등록·취소할 수 있습니다.

### 1. Azure 앱 등록

1. [portal.azure.com](https://portal.azure.com) → **앱 등록** → **새 등록**
2. 이름 입력 후 등록
3. **인증** → **플랫폼 추가** → **모바일 및 데스크톱 애플리케이션**
   - 리디렉션 URI: `https://login.microsoftonline.com/common/oauth2/nativeclient` 선택
4. **API 권한** → **권한 추가** → Microsoft Graph → 위임된 권한
   - `Calendars.ReadWrite`, `User.Read` 추가
5. **개요** 페이지의 **애플리케이션(클라이언트) ID** 복사

### 2. .env에 추가

```
OUTLOOK_CLIENT_ID=복사한_클라이언트_ID
```

### 3. 최초 실행 시 인증

아웃룩 연동을 처음 사용하면 디바이스 코드 인증 안내가 표시됩니다:

```
  🔑 아웃룩 로그인이 필요합니다:
  To sign in, use a web browser to open the page https://microsoft.com/devicelogin
  and enter the code XXXXXXXX to authenticate.
```

브라우저에서 코드 입력 후 Microsoft 계정으로 로그인하면 이후 자동으로 토큰이 갱신됩니다.

## 사용법

```bash
python reserve.py
```

실행하면 대화형 메뉴가 시작됩니다.

```
==================================================
  서울 AI 허브 회의실 예약봇
==================================================

  무슨 작업을 실행할까요?

  1. 회의실 예약
  2. 회의실 예약 취소
  3. 회의실 예약 및 예약번호 조회
  0. 종료
```

---

### 1. 회의실 예약

건물 → 회의실 → 날짜 → 시간 순으로 번호/입력 선택.

```
  1. 회의 건물을 선택해주세요
    1. 서울 AI 허브
    2. 한국교총
    ...

  시설 목록 조회 중...

  2. 회의실을 선택해주세요        ← 선택한 건물의 시설을 자동 조회
    1. 3층 미팅룸1
    2. 3층 미팅룸2
    3. 4층 미팅룸1
    4. 4층 미팅룸2

  3. 예약 날짜를 입력해주세요 (yyyy-mm-dd) :
  4. 예약 시간을 입력해주세요 (예: 09:00-10:30) :
```

- 시간은 30분 단위 (예: `09:00-10:30`)
- 전화 문의 시설 선택 시 안내 메시지 출력 후 종료
- 예약 실패 시 해당 날짜 현황 조회로 바로 이동 가능
- 예약 성공 후 아웃룩 캘린더 등록 여부 확인 (참가자 이메일 입력 시 초대 발송)

---

### 2. 회의실 예약 취소

현재 시각 이후의 예약만 시간순으로 표시. 번호로 선택 후 확인. 아웃룩 일정이 연동된 예약은 취소 시 함께 삭제 여부를 안내합니다.

```
  예약 목록 조회 중...

  1. 취소할 예약을 선택해주세요
    1. 2026-03-10  09:00 ~ 10:00  3층 미팅룸1
    2. 2026-03-12  14:00 ~ 15:30  3층 미팅룸2
    ...

  선택: 2026-03-10  09:00 ~ 10:00  3층 미팅룸1
  정말 취소하시겠습니까? (yes/no) :
```

---

### 3. 회의실 예약 및 예약번호 조회

날짜 입력 → **전체 건물**의 해당 날짜 예약 현황 표시. 우리 예약은 `◀` 표시 및 예약번호 노출.

```
  1. 조회할 날짜를 입력해주세요 (yyyy-mm-dd) : 2026-03-09

  📋 2026년 3월 9일 (월) 예약 현황

  회의실            시간                 예약자                예약번호
  ----------------------------------------------------------------------
  3층 미팅룸1        09:00 ~ 10:00      피아스페이스(주)          00000001962026003461 ◀
  3층 미팅룸1        10:30 ~ 12:00      서울 AI 허브             -
  4층 미팅룸1        14:00 ~ 15:00      네이션에이               -
  회의실 B           09:00 ~ 10:00      피아스페이스(주)          00000001802026001234 ◀
  회의실 B           10:30 ~ 11:30      넥스톰                   -
```

### 예약 성공
```
🔐 로그인 중...
🏢 사이트 선택: 서울 AI 허브
📆 날짜 선택: 2026-03-09
🚪 시설 선택: 3층 미팅룸1
⏰ 잔여시간 05:00
🕐 시간 선택: 09:00 ~ 10:00
⏰ 선택 후 잔여시간 04:00
➡️  예약 정보 확인 페이지로 이동...
📝 예약 신청 중...
🎉 예약이 완료되었습니다!
```

### 예약 불가 케이스
```
❌ 다음 시간은 이미 예약되어 있습니다: 09:00, 09:30
❌ 잔여 시간 부족: 필요 60분, 잔여 30분
❌ '2026-03-08'은 예약 불가능한 날짜입니다 (주말/공휴일 또는 예약 마감)
⚠️  '희경빌딩 A동'은 온라인 예약이 불가하며 담당자에게 전화 문의가 필요합니다.
```
